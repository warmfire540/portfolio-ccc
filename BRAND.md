# Curious Cat Consulting — Brand Guidelines

## Colors

| Token       | Hex       | Usage                                      |
| ----------- | --------- | ------------------------------------------ |
| primary-50  | `#f0f9ff` | Subtle backgrounds, hover states           |
| primary-100 | `#e0f2fe` | Light backgrounds, badges                  |
| primary-200 | `#bae6fd` | Borders, dividers                          |
| primary-300 | `#7dd3fc` | Dark-mode links, secondary accents         |
| primary-400 | `#38bdf8` | Dark-mode interactive elements             |
| primary-500 | `#0ea5e9` | Mid-range accent                           |
| primary-600 | `#0284c7` | **Brand anchor** — buttons, links, accents |
| primary-700 | `#0369a1` | Hover states for primary-600               |

| primary-900 | `#0c4a6e` | Darkest brand color, hero backgrounds      |

### Color Usage

- **Backgrounds:** Light `gray-50` / Dark `gray-900`
- **Text:** Light `gray-800` / Dark `white` or `gray-300`
- **Links & accents:** Light `primary-600` hover `primary-800` / Dark `primary-400` hover `primary-300`

## Typography

- **Font family:** Geist Sans (body), Geist Mono (code/terminal)
- **Weights:** 300 (light), 400 (regular), 500 (medium), 600 (semibold), 700 (bold)
- **Headings:** Bold (700), `gray-800` / `gray-100` in dark mode
- **Body:** Regular (400), `gray-700` / `gray-300` in dark mode
- **Scale:** h1 `text-4xl`, h2 `text-3xl`, h3 `text-2xl`, h4 `text-xl`, h5 `text-lg`, h6 `text-base`

## Components

- **Primary button:** `bg-primary-600 hover:bg-primary-700 text-white rounded-md`
- **Secondary button:** `border border-primary-600 text-primary-600` (dark: `border-primary-400 text-primary-400`)
- **Cards:** `rounded-lg shadow-md` white / dark `gray-800`
- **Focus rings:** `ring-2 ring-primary-500 ring-offset-2`
- **Inputs:** `rounded-md border border-gray-300 focus:ring-primary-500`

## Spacing & Layout

- **Container:** `max-w-4xl mx-auto px-4 sm:px-6 lg:px-8` (or wider for full-width sections)
- **Section padding:** `py-16` or `py-24`
- **Card padding:** `p-6`
- **Button padding:** `px-4 py-2` (small) or `px-6 py-3` (large)

## Voice

Professional, approachable, concise. Avoid jargon; explain technical concepts clearly. Lead with curiosity.

## Identity

- **Company:** Curious Cat Consulting LLC
- **Tagline:** Curiously Better Software
- **Logo:** Cat icon SVG with company name and tagline. Use `fill="currentColor"` for flexible theming.
